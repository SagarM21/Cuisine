# Get Your Meals

A new Flutter project.

## Features

- Bottom Tab feature to switch
- Sidebar
- Favorite Meal Section
- Filtration based on different Meal types
- Multi Screen Application
- Using riverpod to manage state
- Explicit/Implicit animations added
